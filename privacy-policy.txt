PRIVACY POLICY
Pluton Bot

Last updated: 5 February 2026

Pluton respects user privacy and is committed to protecting collected data.

1. Data We Collect
Pluton may collect and store:
- Discord User IDs
- Discord Server IDs
- Vouches (positive and negative)
- User profile data (images, links, colors, products)
- Scam reports and timestamps
- Command usage metadata

Pluton does not collect passwords, email addresses, IP addresses, or payment information.

2. How Data Is Used
Collected data is used to:
- Display user reputation and vouch profiles
- Enable scam detection and reporting features
- Prevent abuse and enforce rules
- Improve performance and security

Data is never sold or shared with third parties.

3. Data Storage & Security
Data is stored securely with restricted access. Reasonable safeguards are used to protect information, but no system can be guaranteed 100% secure.

4. Data Visibility
Some data, including vouches, profiles, and reports, may be publicly visible within servers using Pluton.

By using the Bot, you consent to this visibility.

5. Data Retention
Data is retained as long as it remains relevant to the Bot’s functionality or abuse-prevention systems.

6. Data Removal Requests
Users may request removal of profile data through official support channels. Certain data may be retained to prevent abuse or fraud.

7. Children’s Privacy
Pluton is not intended for users under the age of 13. Any data discovered from underage users will be removed.

8. Changes to Privacy Policy
This Privacy Policy may be updated at any time. Continued use of Pluton indicates acceptance of changes.

9. Contact
For privacy concerns or data requests, contact the official Pluton support server.
